# 共建说明

这个仓库是纯静态网页，没有构建步骤。改完后直接打开 `index.html` 预览即可。

## 文件分工

- `app.js`: 四个人的信息、相册和项目入口
- `index.html`: 页面结构和文案
- `styles.css`: 页面视觉样式
- `README.md`: 项目说明

## 推荐协作方式

1. 从 `main` 拉最新代码。
2. 新建自己的分支，例如 `update-friend-one-card`。
3. 只改自己负责的内容。
4. 本地打开 `index.html` 检查效果。
5. 提交并 push 分支。
6. 在 GitHub 上开 Pull Request。

合并 PR 后，GitHub Pages 会自动更新：

https://qqqzj-crane.github.io/web-all/

## 内容建议

每个人可以先补这些信息：

- 名字或昵称
- 一句话简介
- 三个标签
- 个人链接
- 想展示的照片、项目或记录

如果多人同时改 `styles.css`，最好先在群里说一声，避免样式互相覆盖。
