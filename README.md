# 四人主页

一个独立静态主页，内容集中在四个人的信息、相册和项目入口。没有使用官方游戏图片或改动相邻的 `blog` 项目。

线上地址：

https://qqqzj-crane.github.io/web-all/

## 打开

直接用浏览器打开 `index.html` 即可。

## 修改四个人信息

编辑 `app.js` 开头的 `crew` 数组：

- `name`: 名字
- `role`: 角色称号
- `mark`: 头像圆章里的字
- `quote`: 简介
- `tags`: 标签
- `link`: 个人链接

`memories` 控制相册和项目入口内容。
